# 🌈 عالم الحساب الممتع للأطفال (Math Fun World for Kids)

تطبيق تعليمي تفاعلي، مبهج، ومصمم خصيصاً لمساعدة الأطفال (من سن 5 إلى 7 سنوات) على تعلم أساسيات الجمع والطرح بطريقة مرئية وتفاعلية باستخدام الفواكه والرموز اللطيفة، وبدعم ذكي وتفاعلي متكامل من الذكاء الاصطناعي (Gemini AI)! 🍎🐰🏆

---

## 🎨 مميزات التطبيق (Features)

*   **👾 واجهة تفاعلية مبهجة للأطفال**: تصميم جذاب بألوان زاهية وخطوط مخصصة وحركات انسيابية تناسب الأطفال.
*   **🍎 تمثيل مرئي ممتع للمسائل**: يتم عرض الأرقام على هيئة عناصر رسومية ملموسة (تفاح، موز، أرانب، سيارات...)، مما يسهل على الطفل ربط الأرقام بالواقع.
*   **🧠 ثلاثة أوضاع للعب والتعلم**:
    1.  **تعلم الجمع 🍎➕🍎**: مسائل جمع مبسطة مع تمثيل مرئي.
    2.  **تعلم الطرح 🍌➖🍌**: مسائل طرح مبسطة حيث يتم توجيه الطفل بصرياً.
    3.  **إختبار الذكاء 🏆**: خلط عشوائي بين الجمع والطرح لقياس مهارات الطفل وزيادة مهارة التفكير السريع.
*   **✨ دعم ذكي وتفاعلي من Gemini AI**:
    *   يقوم نموذج الجيل الجديد بالتفاعل بشكل لحظي وحيوي مع إجابة الطفل لإنتاج عبارات تشجيعية مخصصة ومبتكرة (مع الحفاظ على ثلاث كلمات تناسب الأطفال).
    *   **تشغيل آمن ومضمون**: في حالة عدم توفر اتصال بالإنترنت أو عدم تفعيل مفتاح الذكاء الاصطناعي، ينتقل التطبيق فوراً لتقديم رسائل تشجيعية مخزنة محلياً لضمان عدم حدوث أي انقطاع باللعب!

## 🚀 كيفية التشغيل محلياً (How to Run Locally)

لتشغيل هذا التطبيق على جهازك الشخصي، اتبع الخطوات البسيطة التالية:

### 1. المتطلبات الأساسية (Prerequisites)
تأكد من تثبيت بيئة تشغيل **Node.js** (الإصدار 18 أو أحدث) على جهازك.

### 2. تثبيت التبعيات (Installation)
قم بفتح الطرفية (Terminal) في مجلد المشروع ونفذ الأمر التالي لتثبيت المكتبات المطلوبة:

```bash
npm install
```

### 3. إعداد مفتاح الذكاء الاصطناعي (Gemini API Setup)
لقد قمنا بتكوين التطبيق بحيث يعمل مع أو بدون الذكاء الاصطناعي. لتفعيل الميزات الذكية من Google Gemini:
1. انسخ ملف الغلاف `.env.example` إلى ملف جديد باسم `.env.local`:
   ```bash
   cp .env.example .env.local
   ```
2. افتح الملف `.env.local` وأضف مفتاح API الخاص بك من [Google AI Studio](https://aistudio.google.com/):
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

### 4. تشغيل خادم التطوير (Run Dev Server)
قم بتشغيل خادم التطوير لمشاهدة التطبيق حياً في متصفحك:

```bash
npm run dev
```
افتح الرابط المعروض (عادةً `http://localhost:5173`) واستمتع باللعب!

### 5. بناء نسخة الإنتاج (Production Build)
لإنتاج ملفات جاهزة ونظيفة للنشر المباشر:

```bash
npm run build
```
ستجد الملفات الجاهزة في المجلد المُنتج `dist`.

## 🌐 كود النشر على صفحات جيت هاب (GitHub Pages Deployment)

بما أن التطبيق مبني بالكامل على واجهة مستخدم لجهة العميل (React SPA)، يمكنك نشره على صفحات GitHub مجاناً وببضع نقرات:

### الطريقة الأولى: النشر اليدوي عبر المجلد `dist`
1. قم بتنفيذ عملية البناء لإنتاج مجلد `dist`:
   ```bash
   npm run build
   ```
2. يمكنك استخدام منصات مثل **Vercel** أو **Netlify** بربط مجلد `dist` مباشرة، أو دفعه لفرع `gh-pages` في مستودع GitHub الخاص بك.

### الطريقة الثانية: النشر التلقائي باستخدام الأداة (gh-pages)
1. قم بتثبيت حزمة النشر كـ DevDependency:
   ```bash
   npm install -D gh-pages
   ```
2. قم بإضافة الإعدادات التالية في ملف `package.json`:
   * قم بإضافة حقل `"homepage"` في الأعلى:
     ```json
     "homepage": "https://username.github.io/repo-name",
     ```
     *(استبدل `username` باسم مستخدم جيت هاب الخاص بك، وبـ `repo-name` باسم المستودع).*
   * أضف هذه النصوص في قسم `"scripts"`:
     ```json
     "predeploy": "npm run build",
     "deploy": "gh-pages -d dist"
     ```
3. الآن، كلما أردت نشر التحديثات على رابط GitHub الخاص بك، قم بتشغيل الأمر التالي:
   ```bash
   npm run deploy
   ```

## 🛠️ التقنيات المستخدمة (Core Technologies)

*   **React 19** - واجهة مستخدم ديناميكية وقائمة على المكونات.
*   **Vite** - أداة تجميع وبناء فائقة السرعة.
*   **TypeScript** - توفير حماية قوية للأنواع لتجنب أخطاء برمجية في التشغيل.
*   **Tailwind CSS** - تنسيق وتصميم الواجهات بشكل مرن وسريع ومتجاوب تماماً.
*   **@google/genai SDK** - للتفاعل السلس والمستقر مع أقوى نماذج الذكاء الاصطناعي من جوجل (Gemini).
